### Audio Events

This page lists all(?) available audio events in the game.

Audio events are for example used for effects that should play a sound.

Some of these can also be played using sm.audio.play, though only a few of them work.

:::info note
Not all of these events have been tested, thus some of them may not work.

Only a few of these work with sm.audio.play, the rest have to be used with an audio effect.
:::
